# Unfollowers has no third-party dependencies.
# Everything used is part of the Python standard library.
#
# Minimum required: Python 3.9+
#
# The only non-stdlib dependency is tkinter, which ships with the
# standard Python installer on Windows and macOS.
#
# On Linux it may need to be installed separately:
#   Debian / Ubuntu:  sudo apt install python3-tk
#   Fedora:           sudo dnf install python3-tkinter
#   Arch:             sudo pacman -S tk
